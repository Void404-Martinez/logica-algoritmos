# 📝 Exercício 1

## 📌 Enunciado

> Faça um programa que leia o nome de uma pessoa e mostre uma mensagem de boas-vindas.

## 💻 Código

```python id="k8p2rd"
nome = input('Digite seu nome: ')

print(f'Olá {nome}, é um prazer te conhecer')
```

## 📖 Explicação

Este programa utiliza comandos de entrada e saída para interagir com o usuário, lendo um nome e exibindo uma mensagem personalizada.

### 🔎 Fluxo do programa

* `input(...)` → solicita que o usuário digite seu nome
* `nome` → variável que armazena o valor digitado
* `print(...)` → exibe a mensagem de boas-vindas na tela
