# 📝 Exercício 3

## 📌 Enunciado

> Faça um programa que leia o nome e o salário de um funcionário e mostre uma mensagem com essas informações.

## 💻 Código

```text id="n7v3qx"
START
    WRITE("Digite seu nome: ")
    READ nome

    WRITE("Digite seu salário: ")
    READ salario

    WRITE("O funcionário", nome ,", tem um salário de", salario ,"em junho")
END
```

## 📖 Explicação

Este algoritmo utiliza comandos de entrada e saída para interagir com o usuário, lendo informações e exibindo uma mensagem personalizada na tela.

### 🔎 Fluxo do algoritmo

* `START` → inicia o algoritmo
* `WRITE(...)` → exibe mensagens na tela
* `READ nome` → lê o nome do funcionário
* `READ salario` → lê o salário informado
* `WRITE(...)` → mostra a mensagem com os dados do funcionário
* `END` → finaliza o algoritmo
