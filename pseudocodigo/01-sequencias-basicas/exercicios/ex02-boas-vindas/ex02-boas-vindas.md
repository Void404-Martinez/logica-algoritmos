# 📝 Exercício 1

## 📌 Enunciado

> Faça um programa que leia o nome de uma pessoa e mostre uma mensagem de boas-vindas.

## 💻 Código

```text id="m4x9qp"
START
    Write("Digite seu nome: ")
    read nome

    wirete("Olá", nome ,", é um prazer te conhecer!")
END
```

## 📖 Explicação

Este algoritmo utiliza comandos de entrada e saída para interagir com o usuário, lendo um nome e exibindo uma mensagem personalizada.

### 🔎 Fluxo do algoritmo

* `START` → inicia o algoritmo
* `Write(...)` → exibe mensagens na tela
* `read nome` → lê o nome digitado pelo usuário
* `wirete(...)` → mostra a mensagem de boas-vindas
* `END` → finaliza o algoritmo
